About me

Always trying out new things

After I'd begun programming I knew technology would be a part of my career.

Programming: a newfound passion

I started programming in a computational physics class. Once I felt the power of programming, allowing me to do everything I desired, I knew I would turn it into a career. In the short time since I've started, I've worked with Python, C#, Javascript and HTML (to build this site!). I've also used tools like Mathematica/Wolfram Language as well as Julia. When I need to know something to do something, I check it out and learn it in a short time. New tools don't scare me one bit; I'm actively looking for new ones to pick up.
